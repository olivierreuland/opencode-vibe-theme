## Diff Colors (Git/Version Control)

When viewing git diffs or version control changes, these colors are applied:

| Element | Purpose | Dark Theme | Light Theme | Zed | OpenCode |
|---------|---------|------------|-------------|-----|----------|
| **Added** | Text color for added lines | `#4fd6be` | `#1e725c` | ✓ | ✓ |
| **Removed** | Text color for removed lines | `#c53b53` | `#c53b53` | ✓ | ✓ |
| **Context** | Unchanged lines around changes | `#828bb8` | `#7086b5` | - | ✓ |
| **Hunk Header** | Section headers (e.g., "@@ -10,5 +10,6 @@") | `#828bb8` | `#7086b5` | - | ✓ |
| **Highlight Added** | Specific changed text within added line | `#b8db87` | `#4db380` | - | ✓ |
| **Highlight Removed** | Specific changed text within removed line | `#e26a75` | `#f52a65` | - | ✓ |
| **Added Background** | Background fill for entire added line | `#263238` | `#d5e5d5` | ✓ | ✓ |
| **Removed Background** | Background fill for entire removed line | `#3d2832` | `#f7d8db` | ✓ | ✓ |
| **Added Line Number Bg** | Gutter background for added line numbers | `#21313a` | `#c5d5c5` | - | ✓ |
| **Removed Line Number Bg** | Gutter background for removed line numbers | `#33252c` | `#e7c8cb` | - | ✓ |

### Visual Example

In a typical git diff view:

```
Line# │ Code
──────┼────────────────────────────────────
 10   │   function greet(name) {         ← Context: unchanged line (gray text)
      │                                     diffContext color
──────┼────────────────────────────────────
 11   │ -   return "Hello " + name;      ← Removed line:
      │     ^^^^^^^^^^^^^^^^^^^^^^          • diffRemovedBg: light red background
      │            ^^^^^^^^                 • diffAdded: cyan text for whole line
      │                                     • diffHighlightRemoved: bright red for "Hello "
      │                                     • Line number gutter: diffRemovedLineNumberBg
──────┼────────────────────────────────────
 12   │ +   return "Hi " + name + "!";   ← Added line:
      │     ^^^^^^^^^^^^^^^^^^^^^^^^^       • diffAddedBg: light green background
      │            ^^^^^^         ^^^       • diffAdded: cyan text for whole line
      │                                     • diffHighlightAdded: bright green for "Hi " and "!"
      │                                     • Line number gutter: diffAddedLineNumberBg
──────┼────────────────────────────────────
 13   │   }                               ← Context: unchanged line
```

**Key insight**: The "Highlight" colors show the **specific words/characters** that changed within a line, while the base "Added/Removed" colors show the entire line.

**Note**: Diff colors are used by Zed and OpenCode for version control visualization. Ghostty is a terminal emulator and doesn't have built-in diff functionality.

## Usage Guidelines

### Contrast Ratios
The theme is designed with WCAG AA accessibility standards in mind:
- Normal text: minimum 4.5:1 ratio
- Large text: minimum 3:1 ratio

### Semantic Consistency
- Use **warm peach** (`#fab283`) for functions, properties, and links
- Use **soft purple** (`#9d7cd8`) for language keywords
- Use **mint green** (`#7fd88f`) for strings and success states
- Use **soft red** (`#e06c75`) for variables, errors, and regex
- Use **golden yellow** (`#e5c07b`) for types and string escapes
- Use **cyan** (`#56b6c2`) for operators and information
- Use **orange** (`#f5a742`) for numbers, warnings, and booleans
- Use **gray** (`#808080`) for comments and muted text

### Avoiding Color Blindness Issues
The theme is designed to be distinguishable for most types of color blindness:
- Red and green are never the only distinguishing factor
- Important states use multiple visual cues (color + icon/text)
- High contrast between background and foreground

## Alpha Channel Usage

The theme uses 8-digit hex codes (with alpha channel) for transparent overlays and semi-transparent UI elements:
- Selections: `#fab2833D` (dark) / `#3b7dd83D` (light)
- Active line: `#1e1e1e80` (dark) / `#ededed80` (light)
- Overlays and highlights use appropriate alpha values for layering

For opaque colors, 6-digit hex codes are used for clarity.
